# Student Performance Analysis and Career Recommendation System

## 📌 Overview
This project analyzes student academic performance and provides career and stream recommendations based on their marks and interests.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Features
- Analyzed student marks data
- Visualized performance using charts and graphs
- Identified strong and weak subjects
- Suggested career options based on interests
- Recommended suitable stream for next class

## 📈 What I Did
- Cleaned and processed student dataset
- Performed data analysis using Pandas
- Created visualizations for better understanding
- Applied logic to recommend career paths

## 🚀 Outcome
- Helped in understanding student strengths and weaknesses
- Provided data-driven suggestions for future decisions
